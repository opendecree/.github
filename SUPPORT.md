# Getting Help

## Where to ask questions

- **[GitHub Discussions](https://github.com/opendecree/decree/discussions)** — Q&A, ideas, and general conversation
- **[Issues](https://github.com/opendecree/decree/issues)** — bug reports and feature requests only

Please search existing discussions and issues before opening a new one.

## Which repo to file issues in

| Topic | Repo |
|-------|------|
| Server, CLI, Go SDKs, proto, general | [decree](https://github.com/opendecree/decree/issues) |
| Python SDK | [decree-python](https://github.com/opendecree/decree-python/issues) |
| TypeScript SDK | [decree-typescript](https://github.com/opendecree/decree-typescript/issues) |
| Admin GUI | [decree-ui](https://github.com/opendecree/decree-ui/issues) |

Not sure? File it in the [main repo](https://github.com/opendecree/decree/issues) — we'll move it if needed.

## Status

OpenDecree is in **alpha**. APIs, schemas, and behavior are subject to change. Please keep this in mind when evaluating for production use.
